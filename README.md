# Windows End User Troubleshooting Lab

## Overview

This lab contains common Windows troubleshooting scenarios that a Help Desk or IT Support technician might run into. The goal of this project was to build hands-on experience troubleshooting Windows issues and documenting the steps taken to resolve them.

## Scenarios Completed

* Application Would Not Open
* Computer Running Slow
* Network Connectivity Troubleshooting
* Full Disk Space
* Windows Update Troubleshooting
* User Cannot Sign In

## Skills Demonstrated

* Windows troubleshooting
* Application troubleshooting
* Performance troubleshooting
* Basic network diagnostics
* User account troubleshooting
* Windows Update troubleshooting
* Disk space management
* Task Manager analysis
* Command-line troubleshooting
* IT documentation

## Tools Used

* Windows 11
* Task Manager
* Command Prompt
* Windows Settings
* Disk Cleanup
* Windows Update
* Windows Account Settings

## Key Takeaways

This lab helped me get more comfortable troubleshooting common Windows issues using built-in Windows tools. It also gave me practice documenting problems, troubleshooting steps, and resolutions in a way that is similar to real Help Desk work.
